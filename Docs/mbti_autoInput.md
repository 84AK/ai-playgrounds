# MBTI 자동 입력
## 1. 질문데이터 자동 입력
프롬프트를 통해 생성된 12문항 내용은 아래와 같은 형식으로 만들어진다.
---
[E-I: 에너지의 방향]
1번 문제: 쉬는 시간, 옆 반 친구들이 우리 반에 놀러 와 북적거릴 때 나는?

보기1: "오, 무슨 일이야?" 하며 자연스럽게 대화에 섞인다. (E)

보기2: 내 자리에 가만히 앉아 조용히 내 할 일을 한다. (I)

2번 문제: 전교생이 모이는 축제 무대에 오를 기회가 생긴다면?

보기1: 떨리긴 하지만 주목받는 게 즐거울 것 같다. (E)

보기2: 상상만 해도 아찔하다. 뒤에서 응원하는 게 편하다. (I)

3번 문제: 주말에 학교 친구가 갑자기 "지금 나올래?"라고 연락하면?

보기1: "콜!" 바로 준비해서 나간다. (E)

보기2: 미리 약속한 게 아니면 나가기 망설여진다. (I)

[S-N: 정보 인식]
4번 문제: 선생님이 "미래의 우리 학교 모습"을 그려보라고 하시면?

보기1: 지금의 급식실, 교실 구조를 바탕으로 현실적으로 그린다. (S)

보기2: 하늘을 나는 교실, 텔레포트 등 상상력을 총동원한다. (N)

5번 문제: 새로운 초능력을 설명하는 설명서를 읽을 때 나는?

보기1: 단계별 사용법과 주의사항을 꼼꼼히 읽는다. (S)

보기2: 대충 훑어보고 "아, 이런 느낌이구나" 하고 바로 써본다. (N)

6번 문제: 등굣길에 예쁜 꽃을 발견했을 때 드는 생각은?

보기1: "철쭉이네. 색깔이 참 선명하다." (S)

보기2: "이 꽃은 어디서 왔을까? 꽃말은 뭘까?" (N)

[T-F: 판단과 결정]
7번 문제: 친구가 시험을 망쳐서 울고 있을 때 나의 첫마디는?

보기1: "몇 번 문제 틀렸어? 다음엔 오답 노트를 써봐." (T)

보기2: "많이 속상하지? 너 공부 열심히 한 거 내가 다 알아." (F)

8번 문제: 모둠 과제 중 의견 충돌이 일어났을 때 나는?

보기1: 가장 효율적이고 논리적인 의견이 무엇인지 따져본다. (T)

보기2: 최대한 서로 기분이 상하지 않는 방향으로 조율한다. (F)

9번 문제: "너 진짜 똑똑하다!"라는 칭찬을 들었을 때 더 기분 좋은 이유는?

보기1: 나의 능력과 성과를 인정받은 것 같아서. (T)

보기2: 상대방이 나를 좋게 봐주는 마음이 느껴져서. (F)

[J-P: 생활 양식]
10번 문제: 체험학습 가기 전날 밤, 가방을 쌀 때 나는?

보기1: 준비물 리스트를 체크하며 완벽하게 챙겨둔다. (J)

보기2: 내일 아침에 생각나는 대로 대충 챙긴다. (P)

11번 문제: 방과 후 친구들과 놀기로 했을 때 나의 스타일은?

보기1: 1차는 떡볶이, 2차는 코노! 미리 코스를 짜둔다. (J)

보기2: 일단 만나서 그날 기분에 따라 정한다. (P)

12번 문제: 숙제 마감 기한이 다가올 때 나의 행동은?

보기1: 미리미리 계획해서 여유 있게 끝낸다. (J)

보기2: 미루고 미루다 벼락치기로 끝낼 때 집중력이 폭발한다. (P)
---
그럼 이 결과 내용에서 문제과 2개의 보기를 유형별로 파악해서 해당 유형별 question content와 option 1, option2 부분에 입력창에 자동으로 들어갈 수 있게 만들어줘.
