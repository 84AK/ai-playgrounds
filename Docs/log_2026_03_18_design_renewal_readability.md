# 🛠️ 작업 로그 (2026-03-18)

## 📋 개요
사용자 요청에 따라 플랫폼 전체의 구형 다크 글래스 트렌드를 **밝고 친근한 일러스트 기반 팝(Pop) 테마**로 리뉴얼하고, 서브 페이지와 특수 구역들의 가독성 불량 및 레이아웃 단절감 문제를 전수 해결했습니다.

---

## 🔍 문제 현상 (Issue & Error)
1. **서브 페이지 기독성 마비**:
   - `/curriculum`, `/study-lab`, `/course/[weekId]`, `/pose/[weekId]` 페이지 내부 마크다운 등에서 `prose-invert` 및 플레인 `text-white` 성질 노미가 발생하여 흰색 바탕 위에서 글자가 고스트화 됨.
2. **레이아웃 단절감**:
   - 바깥 정적 그라데이션과 본문 간의 부자연스러운 너비 분할 및 부유하는 흰색 백캔버스 존이 가시적으로 감지되어 모던 일러스트 컨셉을 방해함.
3. **로딩화면 및 컴포넌트 유틸리티**:
   - `loading.tsx` 와 `/course` 내 드롭존 패널 마감 처리가 다크 색상에 잠식되어 백업 링크나 트랜지션 시 눈에 피로 유발.

---

## 💡 해결 방안 및 구현 (Resolution)

### 1. 전역 레이아웃 고착화 해제
- **`LayoutClientWrapper.tsx`**:
  - `main` 태그에 `bg-transparent shadow-none`을 확실히 달아주어 브라우저 독립 렌더링을 차단.
  - 고정 배경의 색상 지정자를 물리적 해시 테마 코드(`bg-[#FDFAEF]`) 로 치환하여 일체감 형성.

### 2. Markdown 본문 및 폰트 색상 강제 변환
- **`components/MarkdownContent.tsx`**:
  - 다크모드 하드코딩(`text-white`, `prose-invert`, 어두운 code 스니펫 백그라운드) 을 수색하여 **팝 컨셉 디자이너 지정 색상(`text-[#2F3D4A]`)** 및 상큼하고 마일드한 음영으로 풀체인지.
- **서브 페이지 전수 패치**:
  - 마이 스터디 랩, 코스, 커리큘럼 화면에서 상속 받지 못해 부유하던 단락 라벨들을 리얼 라이트화.

### 3. 유틸리티 로딩 및 업로드 패널 리디바인
- **`app/loading.tsx`**:
  - 심볼 앰블럼의 모서리와 앰비언스를 레트로 보더(`border-[#2F3D4A]`)와 투명 색조로 깔끔하게 조각.
- **`UploadHomework.tsx`**:
  - 드래그 앤 드롭존의 과도한 부모 아웃라인 음영을 전면 배제하고 `.bento-item` 디테일의 하얀색 카드 유니티로 상향 보정.

### 4. 피드백 대응 (UI 폴리싱)
- **`/course/[weekId]`**:
  - 우측에 불필요하게 떠돌며 하단 기능과 중복되던 ‘실습 완료?’ 부유 카드 소멸.
  - 헤더 `My Study Lab` 뒤로가기 링크가 홈화면(`/`) 이 아닌 대시보드 주소(`/study-lab`) 로 직행하도록 세이프 가드 설정.
