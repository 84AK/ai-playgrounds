# 🚀 AI Playgrounds 웹 앱 가이드

AI Playgrounds에 오신 것을 환영합니다! 이 플랫폼은 코딩을 처음 접하는 고등학생들이 웹 브라우저만 있으면 누구나 **인공지능 웹서비스**를 기획하고, 디자인하고, 직접 개발 및 배포까지 해볼 수 있도록 돕는 8차시 통합 교육 플랫폼입니다.

---

## 💡 플랫폼 활용 방법

### 1. 홈 대시보드 (My Study Lab)
홈페이지 첫 화면의 **'MY STUDY LAB'**에서는 나의 현재 학습 진행도를 한눈에 확인할 수 있습니다. 
- **MBTI Maker (1~4주차):** AI 프롬프트와 React로 나만의 MBTI 심리테스트를 만듭니다.
- **Pose Game (1~4주차):** 티처블머신과 Canvas 3D를 이용해 내 동작으로 움직이는 포즈 게임을 만듭니다.
- 진행률 바(Progress Bar)를 통해 내가 얼만큼 과정을 수료했는지 직관적으로 확인하세요.

### 2. 오늘의 수업 시작하기 (실습 환경)
실제 0주차부터 진행되는 개발 결과물을 실습하는 곳입니다.
- **나만의 MBTI 메이커:** 나만의 핵심 성향 질문 12가지를 작성하고 디자인 테마를 고르면 완성된 MBTI 테스트 웹앱이 자동으로 생성됩니다. 생성된 프로젝트는 전시관(Showcase)에 바로 공유할 수 있습니다.
- **AI 포즈 게임 플레이:** 웹캠을 통해 내 움직임을 인식하고 인터랙티브하게 조종하는 게임을 직접 플레이해볼 수 있습니다. 

### 3. 우리 프로젝트 전시관 (Showcase)
전국에서 이 프로젝트를 함께하는 친구들이 만든 재미있는 결과물들이 모여있는 곳입니다.
- `CUSTOM APP`, `MBTI TEST`, `GAME` 등 카테고리별로 색다른 앱들을 직접 체험해 보세요.
- 전시된 카드를 클릭하면 상세 기안서(Dossier) 형태로 프로젝트의 자세한 정보와 체험 링크를 확인할 수 있습니다.

---

## 🛠️ 개발 커리큘럼 미리보기

- **1~4주차 (나만의 코딩 MBTI):** 챗GPT를 활용한 기획부터 UI/UX 디자인 수정, 자바스크립트 로직 연동, 그리고 GitHub 기반의 웹사이트 배포까지 다룹니다.
- **5~8주차 (AI 모션 게임):** 구글 티처블 머신으로 나만의 동작 데이터를 학습시키고 예측 모델을 만들어, 화면의 캐릭터를 내 몸으로 직접 조종하는 게임을 완성합니다.

자세한 주차별 상세 학습 목표와 수업 흐름도는 사이트 하단의 **[Curriculum]** 메뉴에서 확인할 수 있습니다.
