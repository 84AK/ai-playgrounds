# 🚀 AI Playgrounds 웹 앱 가이드

AI Playgrounds에 오신 것을 환영합니다! 이 플랫폼은 코딩을 처음 접하는 학생들이 웹 브라우저만 있으면 누구나 **인공지능 웹서비스**를 기획하고, 디자인하고, 직접 개발 및 배포까지 해볼 수 있도록 돕는 8차시 통합 교육 플랫폼입니다.

---

## 💡 주요 기능 및 활용 방법

### 1. 홈 대시보드 (My Study Lab)
홈페이지 첫 화면의 **'MY STUDY LAB'**은 여러분의 개인 학습 관제탑입니다.
- **실시간 프로그레스:** 내가 현재 몇 주차까지 완료했는지 진행률 바(Progress Bar)를 통해 직관적으로 확인하세요.
- **랭킹 시스템:** 우리 반 친구들의 학습 속도와 성취도를 실시간 랭킹 보드에서 확인하고 동기부여를 얻을 수 있습니다.
- **빠른 시작:** `MBTI Maker`(1~4주차)와 `AI Motion Game`(5~8주차) 실습 페이지로 바로 이동할 수 있습니다.

### 2. 나만의 MBTI 메이커 (1~4주차)
인공지능의 도움을 받아 세상에 하나뿐인 심리테스트 앱을 만듭니다.
- **AI 프롬프트 기획:** 챗GPT를 활용해 기발한 질문과 답변 로직을 구상합니다.
- **디자인 테마 선택:** 내 MBTI 앱의 분위기에 맞는 세련된 테마를 골라 즉석에서 적용합니다.
- **원클릭 배포:** 완성된 프로젝트는 전시관(Showcase)에 바로 공유하여 친구들에게 자랑할 수 있습니다.

### 3. AI 모션 게임 개발 (5~8주차)
내 몸의 움직임을 인식하는 인터랙티브 모션 게임을 만듭니다.
- **티처블 머신 연동:** 구글의 AI 모델 학습 도구로 나만의 동작 데이터를 학습시킵니다.
- **3D Canvas 게임:** 학습시킨 모델을 게임 로직과 연결하여, 화면 속 캐릭터를 내 몸으로 직접 조종하는 3D 게임을 완성합니다.
- **피드백 시스템:** 내가 제출한 과제에 대한 AI 선생님과 연구원들의 피드백을 실시간으로 확인할 수 있습니다.

### 4. 프로젝트 전시관 (Showcase)
전국에서 이 프로젝트를 함께하는 친구들의 창의적인 결과물들을 모아놓은 갤러리입니다.
- **다채로운 카테고리:** `MBTI TEST`, `POSE GAME`, `CUSTOM APP` 등 다양한 장르의 앱을 직접 체험해 보세요.
- **프로젝트 상세 정보:** 전시된 카드를 클릭하면 상세 기안서(Dossier)와 개발 비하인드 스토리를 볼 수 있습니다.

## 🛠️ 커리큘럼 로드맵

- **[STEP 1] 1~4주차 (나만의 코딩 MBTI):** 
  AI를 활용한 기획 → UI/UX 디자인 수정 → 자바스크립트 로직 연동 → GitHub 웹사이트 배포
- **[STEP 2] 5~8주차 (AI 모션 게임):** 
  모션 데이터 학습 → AI 모델 생성 → 게임 엔진 연동 → 최종 프로젝트 완성

상세한 주차별 학습 목표는 하단의 **[Curriculum]** 메뉴에서 일러스트와 함께 확인하세요!
