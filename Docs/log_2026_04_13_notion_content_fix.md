## 🛠 수정 및 보완 내용
1. **페이지별 노션 설정 연동 (`app/[track]/[weekId]/page.tsx`)**
   - **문제 원인:** `getCourseContent` 호출 시 노션 설정값(API Key, DB ID)을 전달하지 않아 항상 기본 데이터(구글 시트)로 폴백되는 문제 발생.
   - **수정 내용:** 
     - 각 페이지 컴포넌트에서 쿠키에 저장된 암호화된 노션 키를 읽어옴.
     - `decrypt` 유틸리티를 사용하여 키를 복호화.
     - `getCourseContent`에 설정값을 정상적으로 전달하여 노션 데이터가 우선적으로 로드되도록 수정.
   - **대상 파일:**
     - `app/mbti/[weekId]/page.tsx`
     - `app/pose/[weekId]/page.tsx`
     - `app/curriculum/[track]/[weekId]/page.tsx`

2. **동적 타이틀 적용**
   - 노션에서 설정한 페이지 제목(Title)이 상단 헤더에 실시간으로 반영되도록 `H1` 태그 로직 수정.

3. **API 에셋/이미지 대응**
   - 노션 내의 이미지 블록과 외부 링크(Google Drive 등)가 마크다운 형식으로 정상 변환되어 출력되도록 `lib/notion.ts` 로직 기반으로 작동 확인.
