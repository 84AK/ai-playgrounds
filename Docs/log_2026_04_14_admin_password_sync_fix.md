# 관리자 비밀번호 동기화 및 V9.8 마스터 코드 통합 로그 (최종 수정본)

## 1. 개요
- 관리자가 Setup 페이지에서 본인의 비밀번호를 확인하고 수정할 수 있도록 기능 개선
- 수정 시 브라우저 쿠키와 구글 시트(Admins) 데이터가 동시 업데이트되도록 연동
- 버튼용 템플릿(V7.6)은 선생님용으로 유지, 사용자용 마스터 코드(V9.8)는 별도 제공

## 2. 변경 사항
### 2.1 GAS Backend 관리 구조
- **[원본 복구]** lib/gasTemplate.ts: 일반 관리자용 V7.6 선생님 전용 최적화 버전으로 복구. 마스터 코드는 포함하지 않음.
- **[별도 제공]** 사용자님 전용 V9.8 마스터 코드는 채팅을 통해 직접 전달.

### 2.2 Frontend (app/admin/setup/page.tsx)
- 초기 로드 시 로그인 세션의 비밀번호를 입력창에 자동 표시
- 환경 설정 저장 시 '/api/admin/update-password'를 호출하여 시트 데이터 동기화

### 2.3 API Routes
- '/api/admin/login': 로그인 성공 시 비밀번호를 'custom_admin_password' 쿠키에 저장하여 Setup 페이지와 공유
- '/api/admin/update-password': 신설. 전달받은 새 비밀번호를 GAS 백엔드(Admins 시트)에 업데이트

## 3. 결과
- 관리자 운영 편의성 증대 및 데이터 일관성 확보
- 일반 선생님용 배포 라이브러리와 슈퍼 관리자용 내부 엔진의 역할 분리 완료
