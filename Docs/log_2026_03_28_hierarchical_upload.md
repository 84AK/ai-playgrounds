## ✨ 신규 구현 사항 (Implement)

- **.zip 파일 업로드 엄격 제한**: 
    - 학생이 과제를 선택할 때 `.zip` 파일이 아니면 즉시 차단하고 경고 모달을 띄우는 로직을 `UploadHomework.tsx`에 구현했습니다.
    - 파일 탐색기에서도 기본적으로 `.zip` 파일만 보이도록 `accept` 속성을 강화했습니다.

- **지능형 계층 폴더 시스템 (GAS)**:
    - 구글 드라이브 내에 `[주차]주차` 폴더와 `[학년반]` 폴더를 자동으로 계층화하여 저장하는 기능을 구현했습니다.
    - 폴더가 이미 있으면 재사용하고, 없으면 실시간으로 생성하여 과제를 분류합니다.
    - 최종 경로 예: `과제루트폴더 > 1주차 > 2학년6반 > 1주차_2학년6반_이름.zip`
