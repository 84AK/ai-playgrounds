## 🔍 배경
- 기존 피드백 시스템이 MBTI Maker(1-4주차)에 고정되어 있어, **Pose Game** 학생들의 과제(`POSE_Week1` 등)를 식별하지 못하는 문제 발생.
- 정답 코드 검색 시에도 "1주차"와 같은 단순 키워드만 사용하고 있어 중복 우려.

---

## ✅ 주요 변경 사항

### 1. 피드백 대시보드 UI (Frontend)
- **코스 선택 탭 추가**: MBTI와 POSE를 버튼 하나로 전환 가능.
- **주차 선택 로직 개선**: 선택된 코스에 따라 "MBTI 1주차", "POSE 1주차"와 같이 명시적으로 표시.
- **파라미터 전달**: 앱스스크립트 호출 시 `course_type`을 명시적으로 전달하여 데이터 정확도 향상.

### 2. 구글 앱스 스크립트 고도화 (v5.2)
- **`findFileBySubfolders`**: `courseType` 파라미터를 추가하여 POSE일 경우 `POSE_Week{n}` 형식으로 검색하도록 고도화.
- **`checkUserStatus`**: 코스별 시트 컬럼 자동 매핑 (MBTI: 1-4, POSE: 5-8).
- **`getReferenceCode`**: 코스 전용 정답 파일을 찾을 수 있도록 파라미터 확장.

### 3. 관리자 로그인 보안 및 UX 강화
- **비밀번호 관리**: `ADMIN_PASSWORD` 환경 변수 연동 및 `.env.local` 가이드 추가.
- **비밀번호 토글**: 로그인 창에 눈 모양 아이콘 추가 (보기/숨기기 가능).
- **로딩 애니메이션**: '입장하기' 클릭 시 스피너(Spinner) 및 로딩 상태 표시로 UX 향상.
- **디자인 폴리싱**: Glassmorphism 스타일 및 진입 애니메이션 적용으로 프리미엄 느낌 강화.
