# 4주차 가이드: 세상에 보여주기 (배포) 🚀

## 1. 수업 목표
- 완성된 결과 페이지(`results.html`)를 구현합니다.
- 친구들에게 공유할 수 있는 실제 인터넷 주소(URL)를 만듭니다.

## 2. 수업 내용

### STEP 1: 결과 보여주기
`js/results.js` 파일을 엽니다.

**[실습 - 내 유형 확인하기]**
3주차에서 보낸 MBTI 결과값(예: "ENTP")을 받아와야 합니다.
AI 질문:
> "URL 쿼리 스트링에서 'mbti'라는 파라미터 값을 가져오는 자바스크립트 코드를 알려줘."

그리고 가져온 MBTI 값에 맞는 설명과 이미지를 `data.js`의 `results` 배열에서 찾아서 화면에 보여줍니다.

### STEP 2: 공유하기 버튼 만들기
내 결과를 카카오톡이나 링크 복사로 친구에게 자랑해봅시다.

**[실습 - 링크 복사]**
> "현재 페이지 URL을 클립보드에 복사하는 버튼 기능을 자바스크립트로 만들어줘. alert으로 '복사되었습니다'도 띄워줘."

### STEP 3: Netlify로 배포하기
내 컴퓨터에만 있는 파일을 인터넷 세계로 올리는 과정입니다.

1. **Netlify Drop** 사이트 접속 (회원가입 필요).
2. 내 프로젝트 폴더(`my-mbti-test`)를 통째로 드래그 앤 드롭 합니다.
3. 1분 뒤 생성된 랜덤 URL을 클릭해서 잘 되는지 확인합니다.
4. (선택) `Site Settings`에서 도메인 이름을 예쁘게 바꿔봅니다.

## 3. 수업 마무리
- 완성된 링크를 단톡방에 공유하고 서로의 MBTI를 테스트해줍니다.
- 4주 동안 고생 많으셨습니다! 이제 여러분은 웹 개발자입니다. 🎉
