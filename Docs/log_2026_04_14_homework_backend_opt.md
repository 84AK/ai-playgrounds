## 🛠️ 상세 수정 내역

### 1. 백엔드 (Google Apps Script) 고도화
- **정밀 파일 교체 로직:** 기존의 단순 인덱스 비교 방식에서 벗어나, 파일명 Stem(`TRACK_WEEK_GRADECLASS_NAME`) 매칭 방식을 도입했습니다.
- **결과:** 수정 제출 시 기존 파일이 정확히 삭제되고 새 파일로 교체됨을 확인했습니다 (12:02 KST 기준 검증 완료).

### 2. 과제 제출 UI (UploadHomework.tsx) 개선
- **프로그레스 바 시인성 강화:** 
  - 흐릿한 파스텔톤 바를 **선명한 블루 그라데이션** 바(+네온 글로우 효과)로 교체했습니다.
  - 진행 단계 메시지의 폰트를 **완전한 검정(text-black)**으로 변경하고 굵게 처리했습니다.
  - 'Ping' 애니메이션을 추가하여 활발히 전송 중임을 시각화했습니다.
- **에러 핸들링 보강:**
  - `postAppsScript` 응답을 1차 검증하고, 실패 시 상세 원인을 담은 팝업 통지 시스템을 구축했습니다.
  - 네트워크 단절 혹은 관리자 설정 오류 시 구체적인 가이드를 제공합니다.
- **리다이렉션 방지:** 제출 성공 후 메인 페이지로 튕기지 않고 현재 위치를 유지하도록 수정했습니다.

### 3. 사용자 경험 (StudyLabPanel.tsx) 최적화
- **동기화 팝업 이식:** My Study Lab 진입 시 데이터가 텅 비어 보이는 문제를 해결하기 위해 `LoadingOverlay`를 적용했습니다.
- **로딩 순서:** 커리큘럼 구조 → 개인 진도 데이터 순으로 동기화가 끝날 때까지 팝업이 유지됩니다.
