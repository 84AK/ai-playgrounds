# 📝 가이드 페이지 시각적 개선 및 콘텐츠 업데이트 로그

**날짜:** 2026-04-12
**수정자:** Antigravity (AI Coding Assistant)

## 🔍 수정 배경
사용자로부터 가이드 페이지의 텍스트가 흰색으로 표시되어 배경과 겹쳐 보이지 않는다는 피드백을 받았습니다. 또한, 최근 업데이트된 시스템 기능(랭킹, MBTI 커스터마이징 등)이 가이드에 반영되지 않아 최신화가 필요했습니다.

## 🛠️ 작업 내용

### 1. UI 스타일 수정 (텍스트 가시성 해결)
- **대상 파일:** `app/guide/page.tsx`
- **변경 사항:** 
    - `ReactMarkdown`을 감싸는 `section` 태그에서 `prose-invert` 클래스를 제거했습니다.
    - `prose-slate` 클래스를 추가하여 라이트 테마 배경에서도 텍스트가 명확하게 보이도록 조정했습니다.
    - `prose-p:text-muted-foreground` 및 `prose-headings:text-foreground` 설정을 통해 프로젝트의 전체적인 톤앤매너를 유지했습니다.

### 2. 가이드 콘텐츠 업데이트
- **대상 파일:** `Docs/guide.md`
- **변경 사항:**
    - **주차별 구성 오류 수정:** MBTI 메이커를 1~4주차, AI 모션 게임을 5~8주차로 정확하게 분류했습니다.
    - **최신 기능 안내 추가:**
        - **My Study Lab:** 실시간 진행률 바 및 랭킹 시스템 활용법 추가.
        - **MBTI Maker:** AI 프롬프트 기획 및 테마 선택 기능 상세 설명.
        - **AI Motion Game:** 티처블 머신 연동 및 3D Canvas 게임 실습 내용 보강.
        - **Showcase:** 프로젝트 상세 기안서(Dossier) 확인 방법 추가.
    - **브랜딩 강화:** 하단에 AK Labs 바로가기 링크(`https://litt.ly/aklabs`)를 추가하여 브랜드 연계성을 높였습니다.

## ✅ 결과 확인
- 가이드 페이지(/guide)에서 모든 텍스트가 배경색(#FDFAEF) 위에서 검은색 회색(#2F3D4A) 계열로 정상 출력됨을 확인했습니다.
- 마크다운 문법이 정상적으로 렌더링되며, 벤토 그리드 스타일의 대시보드와 일관된 디자인을 유지합니다.
