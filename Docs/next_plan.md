# 유저 & 비밀번호 입력 통일
앱 시작시 학생 이름, 학교, 비밀번호를 입력 받아 앱을 사용할 수 있게 하고, 입력한 유저 이름과 비밀번호는 showcase 페이지에서 새로운 결과물을 등록할 때 동일한 유저 이름과 비밀번호로 자동으로 등록될 수 있도록 한다.

# 유저 캐릭터 선택
앱 시작시 입력하는 이름과 학교 비밀번호 입력을 마치면 다양한 캐릭터 이미지를 보여줘 유저가 자신의 캐릭터 이미지를 선택하게 한다.

# 유저 세팅 메뉴
유저는 자신이 입력한 이름, 학교, 비밀번호와 캐릭터를 수정할 수 있다.
이 유저 세팅 메뉴는 당연히 해당 유저 이름과 비밀번호를 입력하여 볼 수 있다.
