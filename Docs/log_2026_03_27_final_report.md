## 🚨 에러 및 장애 해결 (Error & Solve)

1. **AI 분석 CORS 이슈**:
    - **에러**: 클라이언트에서 구글 드라이브 파일을 직접 `fetch`할 때 CORS 정책으로 인해 차단됨.
    - **해결**: Next.js API Route(서버)에서 파일을 다운로드하고 처리하도록 구조를 변경하여 해결.

2. **GAS 데이터 매칭 실패**:
    - **에러**: 시트 이름 오타(`courese_contents`) 또는 컬럼 순서 변경 시 데이터를 찾지 못함.
    - **해결**: 시트 이름 유사도 매칭 로직 및 컬럼 헤더 기반 검색 로직(V3_Final)을 적용하여 '무적 매칭' 구현.

3. **Gemini 모델 지원 중단**:
    - **에러**: `gemini-3.1-pro` 등 존재하지 않거나 지원되지 않는 모델명 사용으로 분석 실패.
    - **해결**: 2026년 기준 최신 안정 버전인 `gemini-2.5-pro`로 교체.
