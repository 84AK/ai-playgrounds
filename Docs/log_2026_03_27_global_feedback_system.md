# 작업 로그: 2026-03-27 글로벌 피드백 및 대시보드 고도화 통합

## 📝 개요
학생들이 과제 제출 상태(파일명 형식 등)를 직관적으로 확인하고, 선생님의 피드백을 놓치지 않도록 시스템을 전면 고도화함.

## ✅ 구현 및 수정 사항

### 1. 글로벌 피드백 알림 시스템
- **`FeedbackOverlay.tsx` 신규 구현**: 로그인 시 피드백이 있으면 모든 페이지에서 모달 노출.
- **`app/layout.tsx` 연동**: 전역 레이아웃에 배치하여 접근성 강화.

### 2. 대시보드 실시간 파일 검증 (Home & Profile)
- **GAS `getProgress` 고도화**: 드라이브 파일을 실시간 체크하여 `verified`(정상), `format_mismatch`(형식 오류) 상태 반환.
- **`StudyLabPanel.tsx` (홈)**: ✅/⚠️ 아이콘 및 학년/반 확인 유도 문구 추가.
- **`HomeworkDashboard.tsx` (프로필)**: 홈 대시보드와 동일한 실시간 검증 로직 및 경고 배너 적용.

### 3. 프로필 편집기 유효성 검사 강화
- **`MyProfileEditor.tsx` 고도화**: 학년, 반 등 필수 정보 누락 시 입력창을 **강렬한 빨간색(Red-600) 애니메이션**으로 강조.
- **검증 로직 완전 동기화**: `checkUserStatus`(업로드 페이지)와 `getProgress`(홈/프로필 대시보드) 두 함수의 정규식을 동일하게 엄격한 기준(학년/반 필수 포함)으로 통일하여 모든 화면에서 일관된 상태(✅/⚠️)를 보여주도록 개선.
- **업로드 시각적 피드백 개선**: 파일 선택 시 "권장사항과 다르다"는 경고 대신 **"자동으로 변환됩니다! ✨"**라는 긍정적인 안내 문구와 함께 부드러운 파란색 정보 박스(Information Box) 스타일을 적용하여 학생들의 불안감을 해소.
- **저장 차단**: 필수 값이 없을 경우 저장을 방지하고 알림(Alert)을 통해 사용자 행동 유도.

### 4. 백엔드(GAS) 및 데이터 구조
- **`Users` 시트 규격 확정**: 8번째(H) 열을 피드백 전용으로 지정.
- **피드백 업데이트 API**: 관리자 페이지에서 H열에 메시지를 작성하도록 수정.
- **로그인 로직 보완**: 데이터 공백 및 타입 이슈 해결 (`getDisplayValues`, `trim` 적용).
