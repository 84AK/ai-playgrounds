# 작업 로그 (2026-03-13) - 제출 시스템 고도화

## 작업 내용
### 1. 학습 페이지(Course) 제출 UI 개선
- `app/course/[weekId]/page.tsx`: 
    - 상단 헤더 및 사이드바에 **'과제 제출하기'** 버튼을 추가하여 접근성을 높였습니다.
    - 모바일 환경에서도 쉽게 제출할 수 있도록 하단 플로팅 버튼을 구현했습니다.
- `app/course/[weekId]/UploadHomework.tsx`: 
    - 제출 폼을 **모달(Modal)** 방식으로 변경하여 학습 흐름을 방해하지 않고 과제를 제출할 수 있게 했습니다.
    - `Glassmorphism 2.0` 디자인을 적용하여 프리미엄한 UI를 제공합니다.

### 2. 파일명 자동 변환 로직 고도화
- 이제 파일 업로드 시 파일명이 다음과 같이 자동으로 변환됩니다:
    - 형식: `YYYYMMDD_[학년]학년[반]반_[이름]_[원본파일명]`
    - 예시: `20260313_2학년6반_홍길동_assignment.zip`
- `readLocalProfile`을 사용하여 사용자의 학년과 반 정보를 정확히 가져오도록 개선했습니다.

### 3. 제출 가이드 및 정책 강화
- **압축파일(.zip) 제출 권장**: 여러 파일을 효율적으로 관리하기 위해 `.zip` 제출 안내를 UI 곳면(모달 본문, 강조 텍스트)에 배치했습니다.
- **가이드 문구**: "20250311_2학년6반_홍길동"과 같은 구체적인 예시를 제공하여 사용자의 혼란을 방지했습니다.

## 해결된 문제 및 기대 효과
- 기존의 복잡한 제출 과정을 '버튼 클릭 -> 파일 선택 -> 제출'의 3단계로 단순화했습니다.
- 학년/반 정보를 파일명에 포함함으로써 관리자가 제출물을 정렬하고 확인하는 시간을 대폭 단축할 수 있습니다.
