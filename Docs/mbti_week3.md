# 3주차 가이드: 숨 불어넣기 (로직 구현) 🧠

## 1. 수업 목표
- 버튼을 클릭하면 다음 질문으로 넘어가는 기능을 만듭니다.
- MBTI 점수를 계산하는 로직을 이해하고 구현합니다.

## 2. 수업 내용

### STEP 1: JavaScript의 기본
- **변수(Variable)**: 값을 담는 그릇 (예: `let score = 0;`)
- **함수(Function)**: 어떤 행동을 하는 기계 (예: `function next() { ... }`)
- **이벤트(Event)**: 클릭 같은 사용자 행동 (예: `onClick`)

### STEP 2: 질문 넘기기 기능 구현
`js/questions.js` 파일을 엽니다.

**[실습 1 - 요소 선택하기]**
HTML에 있는 버튼과 텍스트를 자바스크립트로 가져옵니다.
```javascript
const questionEl = document.querySelector('.question-text');
```

**[실습 2 - 질문 보여주기 함수]**
AI에게 요청해봅니다.
> "자바스크립트 배열에 있는 질문 데이터를 HTML 요소에 텍스트로 넣어주는 renderQuestion 함수를 만들어줘."

### STEP 3: 점수 계산 로직 (여기가 중요해요!)
MBTI는 E/I, S/N 등 4가지 지표가 있습니다.
버튼을 누를 때마다 각 성향의 점수를 저장해야 합니다.

**프롬프트 예시:**
> "사용자가 버튼을 클릭했을 때, 그 버튼에 해당하는 MBTI 타입(E 또는 I 등)을 카운트하는 자바스크립트 로직을 짜줘."

### STEP 4: 결과 페이지로 이동
마지막 12번 질문이 끝나면 `results.html`로 이동해야 합니다. 이때 내가 나온 MBTI 결과도 같이 가져가야 합니다.
> "자바스크립트에서 location.href를 사용해서 페이지를 이동할 때, 쿼리 스트링으로 mbti 값을 같이 보내는 방법 알려줘."

## 3. 도전 과제
