# 작업 로그: 상단 내비게이션 연구소 상태 연동 및 UI 고도화 (2026-04-13)

## 1. 개요
선생님께서 요청하신 대로 '개인 연구소' 표시를 더 명확하게 개선하고, 상단 내비게이션 바(Navbar)에 실시간 상태 배지를 추가하여 사용자가 어떤 환경에 있는지 항상 인지할 수 있도록 개선했습니다.

## 2. 주요 변경 사항

### A. 공통 상태 관리 훅 도입 (`useBackendStatus`)
- `hooks/useBackendStatus.ts`를 신설하여 여러 컴포넌트(Navbar, BackendStatus 등)에서 동일한 커스텀 환경 상태(isCustom, teacherName)를 공유할 수 있도록 구조화했습니다.
- 쿠키 변경 사항을 2초 주기로 감지하여 별도의 새로고침 없이도 UI가 실시간으로 반응합니다.

### B. 상단 내비게이션(Navbar) 배지 추가
- **디자인**: 유저 프로필 옆에 📡 아이콘과 함께 `[선생님 성함] 연구소` 배지를 추가했습니다.
- **애니메이션**: 커스텀 환경 활성화 시 부드러운 Scale & Fade 효과와 함께 나타나며, 상태 표시등(Amber Dot)이 맥박치듯 움직여 활성 상태를 강조합니다.
- **반응형**: 모바일 화면에서는 복잡도를 낮추기 위해 배지를 숨기고, 데스크톱 환경에서만 노출되도록 최적화했습니다.

### C. 하단 플로팅 토스트 문구 및 디자인 개선
- 단순한 '개인 연구소' 대신 `[선생님 성함] 연구소`로 문구를 구체화했습니다.
- 이제 누구의 백엔드에 연결되어 있는지 명확하게 확인할 수 있습니다.

## 3. 해결된 문제 및 UX 향상
- 사용자가 현재 테스트 환경인지 운영 환경인지 굳이 하단까지 보지 않아도 상단에서 즉시 인지할 수 있습니다.
- '누구의 연구소인가'에 대한 모호함을 제거하여 멀티 테넌트(Multi-tenant) 성격의 시트 연결 구조를 명확히 했습니다.
