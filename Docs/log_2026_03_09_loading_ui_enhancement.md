# 작업 로그 - 로딩 UI 시각적 강화 (데이터 연동 표시 개선)

## 일시
2026년 3월 9일

## 작업 역할
Designer(디자이너), Worker(작업자), Doc(서기)

## 기능 개요 (Overview)
- 데이터 동기화가 진행 중임을 사용자가 더 명확하게 인지할 수 있도록 시각적 피드백을 강화했습니다.
- 단순히 기다리는 디자인이 아닌, 시스템이 활발히 데이터를 처리하고 있음을 보여주는 역동적인 요소들을 도입했습니다.

## 상세 구현 내용 (Implementation Details)

### 1. 동기화 상태 배지 (Syncing Badge)
- **StudyLabPanel**: 하단 진도 섹션 우측 상단에 플로팅 배지 추가. "Google Sheet Syncing..." 메시지와 함께 바운스(Bounce) 애니메이션이 적용된 도트 인디케이터가 표시됩니다.
- **HomeworkDashboard**: 마이페이지 과제 섹션 상단에도 동일한 플로팅 배지를 적용하여 통일감 있는 시스템 메시지를 제공합니다.

### 2. 프로그레스 바 스캔 효과 (Scanning Light Effect)
- **globals.css**: `@keyframes progress-scan`을 추가하여 왼쪽에서 오른쪽으로 빛이 훑고 지나가는 광원 효과를 구현했습니다.
- 데이터 연동 중(`isLoadingProgress`)일 때, 모든 진행도 게이지 바 위에 이 스캔 효과가 나타나며 시각적으로 '스캐닝' 중임을 강조합니다.

### 3. 상태 인디케이터 세분화
- 메인 대시보드 탭 옆에 `SYNCING...` 텍스트와 회전하는 스피너 아이콘을 추가하여 아주 작은 공간에서도 상태를 확인할 수 있게 했습니다.

## 결과 및 기대 효과
- **사용자 불만 해소**: 데이터가 언제 갱신되는지 모르겠다던 불안함을 시각적 장치로 완전히 해소했습니다.
- **미적 완성도**: 단순한 로딩 스피너 대신 프리미엄한 스캔 효과를 사용하여 2026년 기준의 세련된 UI/UX를 구현했습니다.
- **일관성**: 메인과 마이페이지 모두 동일한 시각적 언어를 사용하여 학습 플랫폼으로서의 완성도를 높였습니다.
