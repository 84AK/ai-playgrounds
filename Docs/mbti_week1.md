## 2. 수업 내용

### STEP 1: 개발 환경 준비하기
1. **VS Code 열기**: 설치된 VS Code를 실행합니다.
2. **폴더 열기**: 바탕화면에 `my-mbti-test` 폴더를 만들고 VS Code로 엽니다.
3. **보일러플레이트 복사**: 강사님이 주신 초기 코드(boilerplate)를 내 폴더로 복사해옵니다.
4. **Live Server 실행**: `index.html`에서 우클릭 -> 'Open with Live Server'를 눌러 브라우저를 띄웁니다.

### STEP 2: HTML 뼈대 이해하기
- `<!DOCTYPE html>`: "나 이제 HTML 쓸거야!" 라고 선언하는 것.
- `<html>`: 모든 내용을 감싸는 부모.
- `<head>`: 제목, 설정 등 **보이지 않는 정보**.
- `<body>`: 화면에 **보이는 실제 내용**.

**[실습]** `index.html`의 `<title>` 태그 내용을 "내 이름의 MBTI"로 바꿔보세요.
