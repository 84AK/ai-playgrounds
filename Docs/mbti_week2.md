# 2주차 가이드: 나만의 디자인 꾸미기 🎨

## 1. 수업 목표
- HTML 뼈대에 스타일(CSS)을 입히는 방법을 배웁니다.
- AI에게 구체적인 디자인 코드를 요청하는 방법을 연습합니다.

## 2. 수업 내용

### STEP 1: CSS가 뭔가요?
- HTML이 '사람의 뼈'라면, CSS는 '옷과 화장'입니다.
- 선택자(Selector): 누구를 꾸밀지 선택 (예: `body`, `.btn`).
- 속성(Property): 어떻게 꾸밀지 결정 (예: `color: red;`).

### STEP 2: AI에게 디자인 요청하기
디자인을 할 줄 몰라도 괜찮습니다. AI에게 "말로" 디자인을 시켜보세요.

**프롬프트 예시:**
> "MBTI 테스트 시작 버튼을 만들고 싶어. 배경색은 파스텔톤 주황색이고, 모서리가 둥글고, 마우스를 올리면 살짝 커지는 효과가 있는 CSS 코드를 짜줘."

### STEP 3: style.css 채우기
**[실습 1 - 전체 배경]**
AI에게 "웹페이지 전체 배경색을 부드러운 베이지색(#f5f5f5)으로 하고 글씨체는 깔끔하게 해주는 CSS 알려줘"라고 물어보고 `css/style.css`의 `body` 부분에 적용해보세요.

**[실습 2 - 버튼 꾸미기]**
`index.html`에 있는 '시작하기' 버튼(`<button class="btn">`)을 위한 CSS를 AI에게 받아 적용해보세요.

**[실습 3 - 질문 화면 꾸미기]**
`questions.html`을 열고, "질문 텍스트는 크게, 답안 버튼은 세로로 길게 배치하는 Flexbox CSS 코드 알려줘"라고 요청해보세요.

## 3. 핵심 팁
- `class="이름"`을 잘 확인해야 스타일이 적용됩니다.
- 코드가 작동하지 않으면 AI에게 "이 코드가 적용이 안 돼. 혹시 HTML 클래스 이름이랑 맞는지 확인해 줄래?"라고 물어보세요.
