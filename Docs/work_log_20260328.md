## 2. 수정 및 구현 내용

### [쇼케이스 학교 이름 자동 통합 (Normalization)]
- **파일**: `app/showcase/page.tsx`
- **내용**: 
    - 사용자가 제각각으로 입력한 학교 이름(`인천 대건고`, `대건 고등학교`, `인천대건`)을 하나의 표준 이름(`인천대건고등학교`)으로 자동 통합함.
    - 드롭다운 필터에서 중복된 학교명이 사라지고, 클릭 한 번으로 모든 동일 학교 결과물을 조회할 수 있도록 개선함.
    - `고`, `고등`으로 끝나는 명칭을 공백 제거와 함께 `고등학교`로 통일함.

### [MBTI 데이터 로딩 호환성 수정]
- **파일**: `app/mbti/play/page.tsx`, `app/mbti/result/page.tsx`
- **내용**:
    - 구글 스프레드시트 헤더가 대문자(`Text`, `Option1`, `MBTI_Type`)로 되어 있어 생기는 데이터 바인딩 오류 해결.
    - `(data.Text || data.text)`와 같은 방식으로 대소문자 모두 호환되도록 수정.
    - 결과 페이지에서 커스텀 결과가 있는 경우(`Names`, `Description` 등) 우선적으로 표시되도록 반영.

## 3. 발생했던 에러 및 해결 과정
- **문제**: 쇼케이스 카드 클릭 시 "질문 내용이 없습니다" 문구 출력.

- **해결**: 모든 데이터 접근 지점에 대소문자 방어 코드를 적용하여 해결.
